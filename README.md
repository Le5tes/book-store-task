# book-store-task
A REST-API for a book store in express

## Setup
- Clone this repository:
``` $ git clone git@github.com:Le5tes/book-store-task.git ```
- Ensure you have node and npm installed:
    - MAC-OS: 
``` $ brew install npm ```
    - Linux: 
``` $ sudo apt install npm ```

- Install dependencies: 
``` $ npm install ```
